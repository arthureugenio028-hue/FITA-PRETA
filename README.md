# FITA-PRETA
Em uma cidade marcada por segredos, uma simples fita preta começa a aparecer em lugares estranhos, ligada a pessoas que desaparecem sem deixar rastros. Dizem que quem encontra a fita recebe um aviso… ou uma sentença. Entre mistério, traições e perigos escondidos nas sombras, um grupo decide investigar a origem dessa lenda urbana.
